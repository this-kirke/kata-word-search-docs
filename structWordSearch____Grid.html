Detailed Description

This structure contains metadata describing the grid, such as its width and height, as well as a pointer to a Slice containing all of the grid's entries.

A structure defining a word search puzzle grid. Grid entries are arranged in flattened row-major format. That is, rows are placed end to end in memory, packed so that we can calculate the location of each grid coordinate (row, column) by its offset from the start of the entry data.
